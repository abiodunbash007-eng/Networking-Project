By completing this practical exercise, I was able to apply core network security principles to real-world scenarios, strengthening my ability to design secure network environments, identify potential threats, and implement effective mitigation strategies.

The Business Problem
DatNet LTD is currently expanding their operations across continents. The expansion has to be done properly especially as it pertains to their network environment. If not addressed properly, it may lead to Data protection issues and legal repercussions. Therefore, proper network and security simulation is required to identify potential issues and recommend security-driven solutions. The Business Problem
The Problem: Purpose: Build a simulated office network and secure it

The company wants its new departments and office connected through a router and switches using Cisco Packet Tracer.

As part of its internal IT development program, I was assigned as a junior network engineer to design and implement a secure and efficient small office network using Cisco Packet. The network must support day-to-day operations while protecting sensitive internal services from unauthorized access.

My Role I have been engaged as a Junior Network Engineer & Cybersecurity Analyst to support DatNet LTD. My responsibility is to build the network and confirm communication and apply Access Control Lists (ACLs) to limit unnecessary access between departments. I am expected to deliver actionable insights that will help leadership make informed decisions and prevent further operational or financial deterioration.


This project demonstrates how network administrators can use ACLs to enhance security, reduce unauthorized access, and regulate communication paths in a simple routed environment.

I design, configure, and secure a small multi-department network using Cisco Packet Tracer by applying Access Control Lists (ACLs) on the router to manage and control network traffic between departments. 
I help the company understand networks and test and verify access control, service availability, and network security.
